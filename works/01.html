OUTLINEアプリケーションの概要

プログラミングスクールの最終課題制作としてフリーマーケットのアプリケーションを作成しました。
ユーザーを登録すると商品を出品できるようになります。自身が出品した商品は、編集と削除をすることができます。他のユーザーが出品した商品は、クレジットカードを用いて購入することができます。

クレジットカード購入機能については、PAY.JPが提供しているAPIを使用しています。

  • 開発で苦労したこと

    経験が不足しているため、構文ミスやスペルミスなどの細かなミスが頻発し、それを発見し解消するまでに時間がかかってしまうことがありました。幸いにも時間的な余裕は持てていたため、エラーに向き合う時間を儲けることができた点は良かったと思います。

    また、Java Script の理解がまだまだ浅く、本アプリケーションの実装において最も時間がかかってしまいました。その点を解消するために、基本的な知識から独学にて学習をおこなっています。

  • 今後実装したいと思っていること

    Ruby on Rails に関してはある程度の流れを理解できていると感じるので、今後の追加実装やオリジナルアプリケーションの実装においては Java Script を意識的に用いることで、さらなる理解に繋げていきたいと思います。

    フリマアプリに関しては、商品画像のプレビュー機能を Java Script を用いて実装いたしました。
